# MiniShinobi Frontend

React + Vite + Tailwind CSS dashboard for MiniShinobi.

## Development

```bash
npm install
npm run dev
```

The Vite dev server auto-proxies `/api` and `/auth` requests to `localhost:3000` (the backend).

## Production Build

```bash
npm run build
```

Output goes to `dist/` and is served automatically by the Express backend.

## Tech Stack

- **React 19** — UI library
- **Vite 7** — Build tool
- **Tailwind CSS 4** — Utility-first CSS
- **Axios** — HTTP client
- **React Router** — Client-side routing
